# Class Diagram: Строительство Дома Дружбы

## Обзор

Эта диаграмма классов показывает объектно-ориентированную структуру системы строительства "Дома Дружбы".

## Иерархия классов

### Иерархия строителей

| Class | Type | Attributes | Methods |
|-------|------|------------|---------|
| Строитель | Abstract | + сила: int, + имя: String | + строить(дом: Дом) |
| Животное | Abstract | extends Строитель | + найти_друга() |
| Чебурашка | Concrete | + сила = 50, + собрано_кирпичей: int = 0 | extends Животное, + добавить_кирпич(кирпич: Кирпич) |
| Гена | Concrete | + сила = 70, + инструмент: String = "Гармошка" | extends Животное, + играть_на_гармошке(), + передать_кирпич() |

### Иерархия материалов

| Class | Type | Attributes | Methods |
|-------|------|------------|---------|
| Материал | Abstract | + вес: int, + id: int | + использовать() |
| Кирпич | Concrete | + вес = 5, + материал: String = "Глина" | extends Материал |

### Строительный объект

| Class | Type | Attributes | Methods |
|-------|------|------------|---------|
| Дом | Concrete | + количество_кирпичей: int = 0, + статус: СтатусДома = "ПРОЕКТ" | + добавить_кирпич(кирпич: Кирпич), + построить(), + зарегистрировать_друзей() |

### Перечисление

| Enum | Values |
|------|--------|
| СтатусДома | ПРОЕКТ, СТРОИТЕЛЬСТВО, ПОСТРОЕН |

## Связи

- **Чебурашка --> Кирпич**: использует
- **Гена --> Кирпич**: передает
- **Чебурашка --> Дом**: строит
- **Гена --> Дом**: строит
- **Дом --> СтатусДома**: имеет

## Шаблоны проектирования

### Строитель (Builder)
```java
public class Дом {
    public void добавить_кирпич(Кирпич кирпич) {
        количество_кирпичей++;
        if (количество_кирпичей >= 100) {
            статус = "ПОСТРОЕН";
        }
    }
}
Заметки
Чебурашка и Гена — оба Строители, но у Гены есть уникальный метод играть_на_гармошке()

Для начала строительства нужно минимум 10 кирпичей

Для завершения строительства нужно 100 кирпичей

СтатусДома меняется по мере добавления кирпичей

note right of Чебурашка::добавить_кирпич
  **Builder Pattern**
  Каждый добавленный кирпич
  увеличивает счетчик.
  При достижении 100 кирпичей
  дом считается построенным.
end note
