# Activity Diagram: Алгоритм системы "Строительство Дома Дружбы"

## Обзор

Эта диаграмма активности показывает алгоритм работы системы строительства "Дома Дружбы".

## Описание потока

### Шаг 1: Сбор материалов
- Чебурашка и Гена собирают кирпичи
- Проверка: кирпичей >= 10?
- **Нет** - Продолжают сбор

### Шаг 2: Начало строительства
- **Да** - Закладка фундамента
- Начало кладки стен

### Шаг 3: Основной цикл строительства
- Добавить кирпич
- Гена передает кирпич Чебурашке
- Чебурашка укладывает кирпич
- Проверка: кирпичей >= 100?
- **Нет** - Продолжают добавлять кирпичи
- **Да** - Завершение строительства

### Шаг 4: Финал
- Завершить строительство
- Зарегистрировать всех друзей в системе
- Отпраздновать с музыкой на гармошке

## Точки принятия решений

| Условие | Результат |
|-----------|--------|
| Кирпичей >= 10? | Если да → закладка фундамента. Если нет → продолжать сбор |
| Кирпичей >= 100? | Если да → завершение строительства. Если нет → продолжать кладку |

## Диаграмма

![Диаграмма активности](activity.png)

```
@startuml
skinparam conditionStyle inside
title Алгоритм системы "Строительство Дома Дружбы"

start
:Собрать кирпичи;
note right
  Чебурашка и Гена
  собирают материалы
end note

if (Кирпичей >= 10?) then (да)
    #lightblue:Заложить фундамент;
    :Начать кладку стен;
    
    while (Кирпичей < 100?) is (да)
        :Добавить кирпич;
        :Гена передает кирпич Чебурашке;
        :Чебурашка укладывает его;
    endwhile (нет)
    
    #lightgreen:Завершить строительство;
    :Зарегистрировать всех друзей в системе;
    :Отпраздновать с музыкой на гармошке;
else (нет)
    #lightyellow:Продолжать сбор;
    backward
endif

stop
@enduml
