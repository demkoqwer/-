# Use Case Diagram: Строительство Дома Дружбы

## Актеры

| Актер | Описание |
|-------|----------|
| Чебурашка | Главный строитель, собирает кирпичи, находит друзей |
| Гена | Крокодил, играет на гармошке, помогает строить |

## Варианты использования

### Пакет: Строительство Дома Дружбы

| Вариант использования | Описание |
|------------------------|----------|
| Построить дом | Основной сценарий: сбор кирпичей, укладка стен, завершение |
| Поиск друга | Поиск и приглашение друга для участия в строительстве |
| Играть на гармошке | Музыкальное сопровождение процесса |
| Добавление кирпича | Базовое действие по укладке строительного материала |
| Зарегистрировать друзей | Финальная регистрация всех участников в системе |

## Связи

### Актер к варианту использования

- **Чебурашка** и **Гена** выполняют: Построить дом
- **Чебурашка** и **Гена** выполняют: Найти друга
- **Гена** выполняет: Играть на гармошке
- **Чебурашка** и **Гена** выполняют: Добавить кирпич

### Отношения Extend/Include

- **Построить дом** ←← **Зарегистрировать друзей** (`<<extend>>`): при завершении строительства

## Диаграмма

![Диаграмма вариантов использования](use_case.png)
```
@startuml
left to right direction

actor "Чебурашка" as Cheb
actor "Гена" as Gena

package "Строительство Дома Дружбы" {
usecase "Построить дом" as BuildHouse
usecase "Найти друга" as FindFriend
usecase "Играть на гармошке" as PlayAccordion
usecase "Добавить кирпич" as AddBrick
usecase "Зарегистрировать друзей" as RegisterFriends
}

' Чебурашка и Гена строят дом (действие)
Cheb --> BuildHouse
Gena --> BuildHouse

' Чебурашка и Гена ищут друзей (действие)
Cheb --> FindFriend
Gena --> FindFriend

' Гена играет на гармошке (действие)
Gena --> PlayAccordion

' Чебурашка и Гена добавляют кирпичи (действие)
Cheb --> AddBrick
Gena --> AddBrick

' Зависимости внутри системы
BuildHouse <.. RegisterFriends : <<extend>> : при завершении\nстроительства
@enduml
```

## Описание

Эта диаграмма вариантов использования иллюстрирует сценарии строительства «Дома Дружбы» персонажами Чебурашка и Гена:

1. **Чебурашка** и **Гена** начинают **строить** дом (основной сценарий)
2. **Чебурашка** и **Гена** **находят** друзей для участия в строительстве
3. **Гена** **играет** на гармошке для создания праздничной атмосферы
4. **Чебурашка** и **Гена** **добавляют** кирпичи в процессе строительства
5. **Завершение строительства** вызывает **регистрацию** всех друзей в системе (расширение)

Связь показывает, что завершение строительства (условие) расширяет основной сценарий действием регистрации друзей (`<<extend>>`).
